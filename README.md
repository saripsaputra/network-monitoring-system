# Network Monitoring System

Network Monitoring System adalah aplikasi dashboard berbasis web untuk membantu monitoring dan pengelolaan perangkat jaringan melalui tampilan yang sederhana dan mudah digunakan.

## Features

- Dashboard monitoring jaringan
- Monitoring CPU, RAM, dan Storage
- Device Management
- Add, Edit, dan Delete Device
- Device Status, Online, Warning, dan Offline
- Network Traffic Chart
- Server Status Monitoring
- Analytics Dashboard
- Recent Activity
- Notification System
- User Profile
- Settings
- Login dan Logout
- Export Report ke PDF
- Responsive Sidebar

## Technologies

Project ini dibuat menggunakan:

- React
- Vite
- Tailwind CSS
- JavaScript
- React Icons
- jsPDF
- jsPDF AutoTable
- LocalStorage
